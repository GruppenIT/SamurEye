Install the python3-pip package to use pip itself.  Using pip together
with the system python might have unexpected results for any system installed
module, so use it on your own risk, or make sure to only use it in virtual
environments.

root@vlxsam04:~# systemctl status samureye-collector
Unit samureye-collector.service could not be found.
root@vlxsam04:~# systemctl status samureye-telemetry
Unit samureye-telemetry.service could not be found.
root@vlxsam04:~# systemctl status samureye-cert-renew
Unit samureye-cert-renew.service could not be found.
root@vlxsam04:~# cat /var/log/samureye-collector/communication.log
cat: /var/log/samureye-collector/communication.log: No such file or directory
